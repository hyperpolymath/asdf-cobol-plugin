# asdf-cobol

[asdf](https://asdf-vm.com) plugin for [GnuCOBOL](https://www.gnu.org/software/gnucobol/).

## Install

```bash
asdf plugin add cobol https://gitlab.com/hyperpolymath/asdf-cobol.git
asdf list-all cobol
asdf install cobol 3.2.0
asdf global cobol 3.2.0
